# Blog Application Backend (Spring Boot)

## Description
This project is a RESTful backend API for a blogging application.
It supports user management, post creation, categories, and authentication.
The application is built using Spring Boot and follows layered architecture.

## Tech Stack
- Java 11
- Spring Boot
- Spring Data JPA
- Spring Security
- MySQL
- Hibernate
- Maven
- Swagger
- JWT    
## Features
- User registration and login
- Role-based authentication
- Create, update, delete blog posts
- Category management
- API validation and exception handling
## Project Structure
- controller → REST endpoints
- service → Business logic
- repository → Database access
- dto → Data transfer objects
- entity → JPA entities
- config → Security and configuration
